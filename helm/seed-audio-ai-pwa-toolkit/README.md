# Seed Audio AI PWA Toolkit Helm Chart

This chart serves the small Seed Audio AI PWA toolkit as a static web app.

Seed Audio AI links:

- Homepage: <https://seedaud.io/>
- Text to Speech: <https://seedaud.io/text-to-speech/>
- Source: <https://github.com/bbwdadfg/seed-audio-ai-site-kit>

## Install

```sh
helm repo add seed-audio-ai-pwa-toolkit https://raw.githubusercontent.com/bbwdadfg/seed-audio-ai-pwa-helm-charts/main
helm install seed-audio-ai-pwa-toolkit seed-audio-ai-pwa-toolkit/seed-audio-ai-pwa-toolkit
```
