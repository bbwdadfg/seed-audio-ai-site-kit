# Seed Audio AI PWA Toolkit

VS Code-compatible extension with commands for opening Seed Audio AI links:

- Homepage: <https://seedaud.io/>
- Text to Speech: <https://seedaud.io/text-to-speech/>
- Source: <https://github.com/bbwdadfg/seed-audio-ai-site-kit>

This is an unofficial helper. It does not call audio generation APIs.
